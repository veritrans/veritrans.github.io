Integrasi Midtrans dengan Jejualan

Mohon selesaikan langkah di bawah ini terlebih dahulu:

  1. Buat akun di Jejualan. Pilih servis Beta, Gamma, atau Delta untuk menggunakan servis Midtrans.
  2. Daftar akun Midtrans di sini.
  3. Lengkapi form registrasi akun, atau dapatkan bantuan dengan menghubungi tim Activation Midtrans dengan Jejualan - URL Name sebagai subjek dan sebutkan Midtrans Merchant ID yang terdaftar.


Silahkan ikuti langkah-langkah berikut untuk memulai integrasi:

  1. Login ke Merchant Administrator Portal Midtrans

    Anda akan melihat nama online shop Anda, dengan mode Sandbox/Production. Pastikan Anda sedang berada di mode Production.



  2. Pilih Settings - Configuration.


    URL Role Redirect URL
    Payment Notification URL [your-site-url]
    Finish Redirect URL [your-site-url]/store/payment/veritrans/success
    Error Redirect URL [your-site-url]/store/payment/veritrans/failed
    Unfinish Redirect URL [your-site-url]/store/payment/


    Note: Note:
    MMohon lengkapi URL dengan http:// atau https:// saat mengisi Notification URL dan Redirect URL, sesuai konfigurasi web server anda. Jika anda tidak yakin, silakan buka URL website anda di browser, dan cek URL anda menggunakan http atau https.


  3. Pilih Settings - Access Keys

    Copy Merchant ID dan Server Key Midtrans (untuk digunakan di langkah selanjutnya).



  4. Login ke Jejualan Admin Panel dari toko online Anda.



  5. Pilih Konfigurasi - pembayaran.



    Pilih Midtrans field, lalu pindahkan mode dari Tidak Aktif menjadi Aktif. Pastikan tombol sudah berwarna biru.

  6. Masukkan Server Key Production Midtrans. Lalu pilih payment method yang ingin Anda gunakan



    Info Info:

    Anda hanya bisa menggunakan payment method yang Anda register ke Midtrans. Untuk pilihan Credit Card, kami merekomendasi Anda untuk menggunakan 3D Secure.

  7. Klik Simpan.
