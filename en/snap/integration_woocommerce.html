Snap WooCommerce Integration Steps

  1. Plugin Installation
  2. Transaction Test


  • Step 1Plugin Installation
  • Step 2Transaction Test

Requirements

  1. Wordpress v3.9.1 or greater (tested up to v4.5.x)
  2. WooCommerce v2.1.11 or greater (tested up to v2.5.x)
  3. PHP v5.4 or greater
  4. MySQL v5.0 or greater

  5. Midtrans plugin for WooCommerce [ Github | Zip ].

    Info: Info:
    Check this page for complete list of supported Midtrans plugin & library.

Installation Process

Choose one of below.

A. Simple Installation
  1. Login to your Wordpress admin panel.
  2. Go to Plugins menu, click add new. Search for Midtrans-WooCommerce plugin.
  3. Install and follow on screen instructions.
  4. Proceed to Configuration Process below.
B. Manual Installation
  1. Download the plugin from Zip link above.
  2. Extract the plugin, then rename the modules folder as midtrans-woocommerce.
  3. Upload the unzipped plugin folder to your WordPress installation's wp-content/plugins/ directory.
  4. Install and activate the plugin from plugins menu within the WordPress admin panel
  5. Proceed to Configuration Process below.

Configuration Process

  1. Go to WooCommerce - Settings - Checkout - Midtrans menu, fill the configuration fields.

    • Fill Title with text button that you want to display to customer
    • Select Environment. Sandbox for testing transaction and Production for real transaction
    • Fill Client and Server key. You can find this credential on Midtrans MAP Dashboard.


  2. Login to MAP portal to configure few options below:

    URL Role Redirect URL
    Payment Notification URL [your-site-url]/?wc-api=WC_Gateway_Midtrans
    Finish Redirect URL [your-site-url]/?wc-api=WC_Gateway_Midtrans
    Error Redirect URL [your-site-url]/?wc-api=WC_Gateway_Midtrans
    Unfinish Redirect URL [your-site-url]/?wc-api=WC_Gateway_Midtrans


    Note: Note:
    Please make sure to input http:// or https:// when filling Notification URL and Redirect URL, according to your web-server configuration. If you are not sure, try opening your web URL in a browser, and check the URL is http or https on the address bar.


Your customer now will be able to pay via Midtrans with multiple available payment methods.

Note: Note:
Please check our Snap-Woocommerce Repo Wiki for more advanced tutorial.
