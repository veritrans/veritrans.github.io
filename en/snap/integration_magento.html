Snap Magento Integration Steps

  1. Plugin Installation
  2. Transaction Test


  • Step 1Plugin Installation
  • Step 2Transaction Test

Requirements

  1. An online store with Magento infrastructure. This plugin is tested with Magento v1.9.
  2. PHP v5.4 or greater.
  3. MySQL v5.0 or greater.

  4. Midtrans plugin for Magento [ Github | Zip ].

    Info: Info:
    Check this page for complete list of supported Midtrans plugin & library.

Installation Process

  1. Download and extract the plugin from this repository.
  2. Locate the root Magento directory of your shop via FTP connection.
  3. Copy the app, lib and skin folders into magento root folder.
  4. In your Magento admin area, Go to System - Configuration menu.

  5. In Sales-Payment Methods option, fill the configuration fields.

    • Fill Title with text button that you want to display to customer.
    • Select Environment. Sandbox for testing transaction and Production for real transaction.
    • Fill Client and Server key. You can find this credential on Midtrans MAP Dashboard.


  6. Login to MAP portal to configure a few options below:

    URL Role Redirect URL
    Payment Notification URL http://[your web]/snap/payment/notification
    Finish Redirect URL http://[your web]/snap/payment/response
    Error Redirect URL http://[your web]/snap/payment/response
    Unfinish Redirect URL http://[your web]/snap/payment/response


Note: Note:
Please check our Snap-Magento Repo Wiki for other advanced features tutorial.


  1. Perform successful transaction in your online store by inputing a dummy credit card number as followed (Sandbox Mode):

    Type Visa
    Card Number 4811 1111 1111 1114
    CVV 123
    Exp. Month 01
    Exp. Year 2020


  2. Examine a few points below to ensure plugin is installed and performs properly.

    Check Point Expected Result When Unexpected Result Occurs
    Check order status in OpenCart back end Order status turns into processing Check endpoint/payment notification URL setting in MAP
    Merchant receives email notification Receive notification Check MAP menu setting - email notification
    Customer receives email notification Receive notification Check MAP menu setting - email notification
